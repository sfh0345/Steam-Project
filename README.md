## About the project

Dit project richt zich op de ontwikkeling van een visueel dashboard voor het gamegedrag van vrienden op Steam. De opdracht is om een basisapplicatie met een dashboard te bouwen. De verplichte onderdelen omvatten het creÃ«ren van een basisapplicatie in Python, het laden van data via de Steam-API, en het tonen van enkele elementen op een GUI op basis van AI-algoritmen. Bovendien wordt er een koppeling gerealiseerd tussen het dashboard en TI voor de toepassing van hardware.
